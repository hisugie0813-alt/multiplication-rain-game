# 구구단 산성비 게임 🌧️

초등학교 2학년 학생들이 곱셈구구를 재미있게 학습할 수 있는 교육용 게임입니다.

## 기능

- 🎮 직관적인 게임 플레이 (물방울 낙하 방식)
- 📊 3단계 난이도 (쉬움/보통/어려움)
- 🎯 특정 단 집중 연습 모드
- 💖 생명 시스템 (3개)
- 🏆 점수 시스템 및 최고 점수 저장
- 🎵 효과음 및 시각 피드백
- 📱 반응형 디자인 (PC, 태블릿, 모바일)

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173`을 열어주세요.

### 빌드

```bash
npm run build
```

### 미리보기

```bash
npm run preview
```

## 기술 스택

- React 18
- TypeScript
- Vite
- CSS3 (애니메이션)

## 게임 방법

1. 난이도를 선택하거나 특정 단을 선택합니다
2. 하늘에서 떨어지는 곱셈 문제를 봅니다
3. 숫자 버튼으로 답을 입력하고 확인을 누릅니다
4. 정답을 맞추면 물방울이 사라지고 점수를 얻습니다
5. 물방울이 바닥에 닿으면 생명이 감소합니다
6. 3개의 생명을 모두 잃으면 게임이 종료됩니다
